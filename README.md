# dotfiles

## 만든 이유 

* windows와 linux에서 동일한 설정파일을 사용하고 싶다.
* 내 PC가 아닌곳에서 작업할 때 원래 사용하던 개발 설정을 가지고 오고 싶다.
* 개발 설정을 안전한 곳에 보관하고 싶다.

## 의존성

* `.ps1` 파일은 PowerShell 5.1 기준으로 작성함
